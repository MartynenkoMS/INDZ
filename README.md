# Метеосервіс

Імітаційний прототип системи метеорологічного сервісу.

## Компіляція

Для компіляції програми виконайте наступну команду в терміналі:

```bash
javac -encoding UTF-8 src/*.java
```

## Запуск

Для запуску програми виконайте:

```bash
java -cp src Main
```

## Структура проєкту

- `src/` - каталог з вихідним кодом
  - `Main.java` - головний клас програми
  - `УчасникМетеосистеми.java` - абстрактний клас учасника системи
  - `IПланСповіщень.java` - інтерфейс для роботи зі сповіщеннями
  - `Метеоролог.java` - клас метеоролога
  - `СервісСповіщень.java` - сервіс для відправки сповіщень
- `docs/` - документація проєкту
